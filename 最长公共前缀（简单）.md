# 最长公共前缀（简单） 
![](./pic/最长公共前缀题目.png)

代码如下：
```
var longestCommonPrefix = function(strs) {
    strs.sort();
    var str='';
    var i=0;
    if(strs.length===0){
        str='';
    }else{
        while(strs[0][i]!==undefined&&strs[0][i]===strs[strs.length-1][i]){
            str+=strs[0][i]
            i++;
        }
    }
    return str;
};
```
