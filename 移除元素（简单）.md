# 移除元素（简单） 
![](./pic/移除元素题目.png)

代码如下：
```
var removeElement = function(nums, val) {
    var i = 0,
        length=nums.length;
    while(i<=length-1){
        while(nums[i]===val){
            nums.splice(i,1)
        }
        i++;
    }
    return nums.length;
};
```
