# 删除排序链表中的重复元素（简单） 
![](./pic/删除排序链表中的重复元素题目.png)

代码如下：
```
var deleteDuplicates = function (head) {
    var current=head;
    while(current!==null&&current.next!==null){
        if(current.val==current.next.val){
            current.next=current.next.next
        }else{
            current=current.next
        }
    }
    return head
};
```
